────────────────────────────  ENGLISH  ────────────────────────────

AI Translator is a menu‑bar translator for macOS that works with any
OpenAI‑compatible API (OpenWebUI, OpenAI, Ollama, OpenRouter, and
similar gateways).

INSTALL
  1. Drag “AI Translator” onto the Applications folder.
  2. Launch it from Applications. The icon appears in the menu bar.
  3. On first launch macOS will ask for Accessibility permission —
     this is required for in‑place translation (simulated ⌘C / ⌘V).

FEATURES
  • Translate between 16 languages with automatic source detection.
  • Connect to any OpenAI‑compatible model through profiles.
  • Custom prompts for different styles (literal, literary, technical…).
  • Quick translate of selected text:
      – in a window, or
      – in place (replaces the selection in the active app).
  • Auto‑swap direction by detected language (e.g. ru ↔ en).
  • Translation history (last 100) with search.
  • API tokens are stored in the system Keychain.
  • Bilingual interface (English / Русский), switchable in Settings.

DEFAULT HOTKEYS
  Open translator window ............ ⌘O
  Open settings ..................... ⌘,
  Quick / in‑place translate ........ ⌘⇧T (configurable)
  Cancel in‑place translation ....... press the same hotkey again

REQUIREMENTS
  macOS 14 (Sonoma) or newer.

NOTE
  This build is signed with an Apple Development certificate for
  personal use. On a Mac where it was not built, Gatekeeper may warn
  on first launch: right‑click the app → Open → Open.


────────────────────────────  РУССКИЙ  ────────────────────────────

AI Translator — переводчик для macOS, живущий в строке меню и
работающий с любым OpenAI‑совместимым API (OpenWebUI, OpenAI, Ollama,
OpenRouter и аналогичными шлюзами).

УСТАНОВКА
  1. Перетащите «AI Translator» в папку Applications.
  2. Запустите из Программ. Иконка появится в строке меню.
  3. При первом запуске macOS попросит разрешение «Универсальный
     доступ» (Accessibility) — оно нужно для перевода на месте
     (симуляция ⌘C / ⌘V).

ВОЗМОЖНОСТИ
  • Перевод между 16 языками с автоопределением исходного.
  • Подключение к любым OpenAI‑совместимым моделям через профили.
  • Кастомные промпты под разные стили (буквальный, литературный,
    технический…).
  • Быстрый перевод выделенного текста:
      – в отдельном окне, или
      – на месте (заменяет выделение прямо в активном приложении).
  • Автоопределение направления по языку (например, ru ↔ en).
  • История переводов (последние 100) с поиском.
  • API‑токены хранятся в системном Keychain.
  • Двуязычный интерфейс (English / Русский), переключается в настройках.

ГОРЯЧИЕ КЛАВИШИ ПО УМОЛЧАНИЮ
  Открыть окно переводчика .......... ⌘O
  Открыть настройки ................. ⌘,
  Быстрый перевод / на месте ........ ⌘⇧T (настраивается)
  Отмена перевода на месте .......... повторное нажатие той же клавиши

ТРЕБОВАНИЯ
  macOS 14 (Sonoma) или новее.
